# CornersGame
# Игра «Уголки"

Игра вдвоем за одним компьютером (hot seat)

Среда разработки: Unity 2019.4.0

## Правила игры: 
После расстановки фигур на шахматной доске, каждый игрок ходит по очереди. Во время хода, он может переместить одну фигуру. Побеждает игрок, который расставит все свои фигуры на место фигур противника.

Расстановка фигур в начале игры:![StartPos](https://github.com/true-10/CornersGame/blob/main/startPos.jpg)

## Правила движения фигур (выбирается в UI, перед стартом игры): 

- фишка может перепрыгивать через другую как в шашках, по диагонали. 
- фишка перепрыгивает по вертикали и горизонтали. 
- фишка не может перепрыгивать, а делает только один шаг в любом направлении.

## UI: 

- выбор правила перемещения перед стартом игры; 
- отображение в HUD номера хода каждого из игроков в текущей партии; 
- перезапуск партии; экран завершения партии с указанием победителя.

## Ресурсы: 
Префабы фигур загружаются из ресурсов (Resources) проекта.

## Управление: 

- нажатие ЛКИ на фигуру игрока - выделение 
- нажатие ЛКМ на свободную клетку - перемещение фигуры, если это возможно. 
- нажатие ПКМ - завершить ход, если это возможно.
